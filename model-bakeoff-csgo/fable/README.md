# 迷你 CS:GO（Counter-Strike 复刻）

用 Three.js 实现的第一人称战术射击，向《反恐精英：全球攻势》致敬的迷你复刻。
纯前端单页，零构建、零安装、零外部资源（地图几何、贴图、人物、枪模、音效全部程序化生成）。

## 怎么玩

直接用浏览器打开 `index.html` 即可（需联网加载 Three.js CDN）。
如果浏览器对 `file://` 有限制，也可以起一个本地静态服务器：

```bash
cd fable
python3 -m http.server 8000
# 打开 http://localhost:8000
```

进入页面后点击「T 恐怖分子」或「CT 反恐精英」选择阵营即开始（自动锁定鼠标）。

## 规则

- **4 v 4**：你 + 3 名队友 BOT 对阵 4 名敌方 BOT，先赢 **8 回合** 获胜。
- **半场换边**：一方先拿到 **4 回合** 时触发换边——T↔CT 阵营互换（比分随队伍带走）、所有人经济重置为 $800 手枪局。
- **回合流程**：购买时间 12 秒（冻结移动）→ 对战 1 分 55 秒 → 回合结算 5 秒。
- **胜负条件**：歼灭对方全员 / T 安放的 C4 爆炸（40 秒倒计时）/ CT 拆除 C4 / 时间耗尽则 CT 胜。
- **C4**：每回合随机一名 T 携带（HUD 有提示），安放 3.2 秒、拆除 6 秒（买了拆弹钳 3 秒）；携带者阵亡后 C4 转交最近的存活 T。
- **死亡观战**：阵亡后自动切到存活队友的第一人称视角（左键切换目标，无法开枪），回合结束复活。
- **经济**：首回合 $800；击杀 +$300（AWP +$100、刀 +$1500）、安放/拆除 +$300；赢回合 +$3250，输回合 +$1400 起连败递增（最多 +$2900）；上限 $16000。
- **存活奖励**：回合存活者保留枪与护甲，阵亡者下回合重置为手枪。

## 功能清单

- **第一人称控制**：PointerLock + WASD、跳跃、下蹲（速度与精度差异）、重力、AABB 逐轴扫掠碰撞 + 自动上台阶，不会穿墙
- **Dust II 灵感地图**：T/CT 出生点、中路（Mid）、A 大道（Long）、A 小道（Short）、B 通道（隧道带顶棚与点光）、A 点高台（多层几何 + 台阶）、B 点仓库；渐变天空穹顶 + 太阳圆盘 + 沙色远雾 + 暖阳光照；中路石板 / 隧道暗土 / A 大道红沙分区地面材质帮助辨路；地上黄圈 A/B 标记 + 空中漂浮 A/B 字母；「中门 / 中路 / B 隧道 / A 大道 / A 小道 / B 门 / A 坡道」中英双语路牌；砖墙/沙地/木箱贴图全部 Canvas 程序化生成
- **半场换边**：先到 4 回合触发——阵营互换、比分随队伍带走、经济重置手枪局、BOT 换皮换名，HUD 横幅「换边」提示
- **迷你雷达**（左下）：地图俯视底图随玩家朝向旋转、居中玩家箭头、队友蓝/黄点、敌人红点（队友视野可见或 2 秒内开过枪才显示，超出范围钉在雷达边缘）、A/B 点位字母、己方 C4 携带者橙点、已安放 C4 红色闪烁方块
- **计分板**：按住 `Tab` 查看两队 击杀/助攻/阵亡/金钱/存活，C4 携带者有标记；助攻 = 死亡前 6 秒内对死者造成过伤害
- **武器系统**：射线命中（世界 AABB + 角色盒，爆头区判定）、基础扩散 + 移动/空中/连发扩散 + 蹲姿加成、**AK/M4 确定性喷射弹道**（上抬 + 水平摆动 + 停火复位，可压枪）、视角后坐力上抬与回落、换弹动画、切枪动画、走路摆枪、枪口火光、弹道曳光（对象池）、命中反馈（Hitmarker + 音效）
- **武器掉落与拾取**：任何角色阵亡掉落主武器（保留弹药，带漂浮名牌），玩家点按 `E` 换枪、无主武器时走过去自动捡，手枪 BOT 也会顺路捡枪；回合开始清场
- **投掷物**：HE 手雷 / 闪光弹 / 烟雾弹 / 燃烧瓶（T）/ 燃烧弹（CT）——弧线弹道、落点反弹（带反弹声）、HE 范围伤害 + 爆炸白闪、闪光弹按距离与朝向致盲（玩家白屏渐退 + 耳鸣，BOT 短暂失明丢失目标）、烟雾弹展开约 15 秒的烟云（多球体积感渲染）、火焰弹落地成 7 秒火区（锥形火苗集群 + 点光 + 烧痕，每 0.25 秒结算伤害）；烟雾会阻断 BOT 视线与闪光、还能浇灭火焰；手持投掷物显示虚线弧线落点预览；回合开始自动清场
- **命中反馈**：方向性血液喷溅、墙面弹孔贴花（48 个对象池，10 秒后淡出）、Hitmarker（击杀变红）+ 音效、命中点飘伤害数字（爆头带 ✕ 金色）
- **死亡观战**：阵亡 1.4 秒后自动跟随存活队友第一人称视角，左键切换目标，HUD 显示「观战: 名字」；无存活队友则停留原地视角；无法开枪 / 移动
- **无线电**：`Z/X/C` 快捷语音显示在左下语音条（X/C 同时在雷达上打出扩散黄圈标点）；队友 BOT 遭遇敌人 / 听到枪声支援 / 下包时也会报点（只显示己方语音）
- **BOT AI**：路点图 + Dijkstra 寻路、视线检测（墙体/烟雾/致盲三重判定）、反应延迟、战斗走位（横向拉扯/控距）、点射节奏、卡住自动跳；每回合随机进攻路线（分散推进不挤一条道）、CT 在点位附近架枪盯入口、听到枪声会赶去支援补位；T BOT 会带包去 A/B 安放并守包，CT BOT 会回防拆包（有拆弹钳 3 秒）；不敢穿越火区（着火立刻逃离，拆包/下包告急时例外）；经济上先枪后甲再道具（CT 优先拆弹钳），闪光 > HE > 火焰 > 烟雾，只有富裕时才买 AWP；战斗中会丢闪光/火焰/HE
- **回合系统**：购买冻结 → 对战 → 结算；歼灭/爆炸/拆除/超时四种胜负；比分与回合数、双方存活数显示；先到 4 回合半场换边（经济重置手枪局），先到 8 回合结束比赛；比赛结束显示 MVP 领奖台（前三名 K/助攻/D）
- **CS2 风格 HUD**：顶部队伍比分药丸（CT 蓝 / T 金橙顶边）+ 中央计时；深色半透明方形面板（血量/护甲/金钱/弹药）；血量下方白色血条；右下**武器快捷栏**（1-4 槽位、选中高亮、显示弹数）；击杀信息流为深色圆角药丸（自己参与的击杀红框）；可调常量的细十字准星（间隙/粗细/描边 CSS 变量）；**全屏购买菜单**（枪械/装备/投掷物三栏卡片，数字键编号不变）；圆形软边雷达（CT 蓝白 / T 金橙点位）；冻结时间最后 3 秒**大数字倒计时**；加粗分段式安放/拆除进度条（含百分比）；**受击方向指示**（屏幕边缘红弧指向伤害来源）；轻量后期（暗角 + 冷暖色分级叠层）；DOM 写入做值缓存避免逐帧刷新
- **CS2 风格画面**：ACES 电影级色调映射 + 提高曝光（画面更透亮不发闷）、更亮更远的沙色雾；**廉价接触阴影**——墙根/箱底一圈乘法混合的渐变暗带（单次合批绘制），近似环境光遮蔽；人物剪影更清晰——CT 蓝灰作战服 + 头盔护目镜、T 橄榄棕 + 头巾；枪模采用更深的金属/聚合物材质，视角转动时武器有轻微滞后摆动（sway）
- **音效**：WebAudio 全程序化合成——每把枪不同的枪声（USP 消音、AWP 低频回响）、脚步（含远处敌人脚步衰减）、换弹、买枪、拔雷针、手雷反弹、HE 爆炸、闪光爆响与耳鸣、烟雾嘶嘶声、火焰点燃/噼啪/熄灭、无线电滴声、C4 加速蜂鸣、拆弹滴答、爆炸音效，无任何音频文件；**现代化回合音**——冻结倒计时提示音、双音上行开局音、CT（冷色上行）/ T（铜管感）不同的回合结束定音，落败听到闷化版本；**简易声音遮挡**——隔墙的枪声/脚步衰减到 35%

## 技术说明

- Three.js **r169**（满足 r160+ 要求），通过 jsDelivr CDN 以 ES Module + importmap 加载；游戏逻辑为普通脚本挂在 `window.CS` 命名空间下，因此 `file://` 直开也能运行。
- 地图采用「开凿式」构建：先声明可通行矩形区域，其余空间自动填充为墙体并贪心合并成大 AABB，网格数少、碰撞盒即渲染盒，射击命中与移动碰撞共用同一套数据。
- 角色与子弹命中都基于解析式射线-AABB（slab 法），不依赖 Three.js Raycaster 遍历三角形，7 个角色 + 全图碰撞盒逐帧检测无压力。
- 无任何后端、无 npm install、无构建步骤、无外部图片/模型/音频资源。
