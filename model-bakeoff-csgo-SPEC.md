# 任务：用 Three.js 做一个可玩的「CS:GO」迷你复刻

目标：尽量像 Counter-Strike: Global Offensive，第一人称战术射击，不是别的射击游戏主题。

## 必须有（缺一不可）
1. 第一人称视角（WASD 移动，鼠标视角；PointerLock；空格跳、Shift 蹲）
2. CS 风格准星、弹药、血量、护甲、金钱、回合计时 HUD
3. 至少一套 Dust II 灵感地图（多层几何：出生点、中路、炸弹点 A/B 可辨认）
4. 武器射击：射线命中、后坐力/扩散、弹匣换弹、切换武器
5. 至少这些枪：Glock / USP、AK-47、M4A4、AWP、刀
6. T / CT 阵营选择与回合制（买刀时间 → 对战 → 回合结算）
7. 经济与购买菜单（`B` 键）：买枪/护甲，用金钱扣款
8. C4：T 在炸弹点安放，CT 拆弹；爆炸/拆弹结束回合
9. 简易 BOT 对手（寻路粗略即可，会移动/射击）
10. 击杀反馈（击杀提示 / 击杀信息）
11. 碰撞：墙体 AABB，不能穿墙；落地与重力
12. 单页可玩：index.html + JS 模块 + style.css + README.md（中文）
13. Three.js 用 CDN（r160+），打开就能玩，无构建步骤、无后端、无 npm install

## 加分
- 喷火/弹道曳光、枪口火光、枪托视角武器模型
- 脚步声、枪声、爆炸声（WebAudio 合成，无音频文件）
- 烟雾弹 / 闪光弹（至少一种投掷物）
- 雷达提示、队伍击杀/存活人数
- 站立/蹲下速度与准确度差异（跑打更散）

## 禁止
- 不要做成堡垒之夜 / COD 大逃杀等其他主题
- 不要后端、不要 npm install
- 不要依赖外部图片/模型资源（几何与材质程序化即可）
