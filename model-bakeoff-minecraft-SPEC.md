# 任务：用 Three.js 做一个可玩的「我的世界」迷你复刻

目标：尽量像 Minecraft（我的世界），第一人称方块沙盒，不是别的主题世界。

## 必须有（缺一不可）
1. 第一人称视角（WASD 移动，鼠标视角；PointerLock）
2. 体素方块世界（至少草地、泥土、石头、木头 4 种方块）
3. 程序生成简易地形（小山+平地即可，别整片平地）
4. 左键破坏方块 / 右键放置方块（准星对准的方块）
5. 简单背包/热键栏：数字键 1-4 切换放置方块类型
6. 天空+光照（白天即可）
7. 物理：有重力、有碰撞，不能穿墙掉出世界
8. 单页可玩：index.html + app.js(+可选模块) + style.css + README.md（中文）
9. Three.js 用 CDN（three r160+），可用 Orbit 以外的 PointerLockControls（CDN）
10. 打开就能玩，无构建步骤

## 加分
- 破坏有粒子/声音（可选）
- 水面
- 树
- 十字准星 UI
- 方块高亮轮廓

## 禁止
- 不要做成港口/城市/别的主题
- 不要后端、不要 npm install
