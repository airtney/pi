# 方块世界

一个无需构建、打开即玩的 Three.js 第一人称体素沙盒。世界包含程序生成的平地、山丘、树林和可编辑方块，并提供基础重力与方块碰撞。

## 开始游戏

直接用现代浏览器打开 `index.html`，保持网络连接以从 CDN 加载 Three.js。点击“进入世界”锁定鼠标；按 `Esc` 可释放鼠标并暂停。

如浏览器限制本地页面加载 ES 模块，可在本目录启动任意静态文件服务器后访问，例如：

```bash
python3 -m http.server 8080
```

## 操作

| 操作 | 按键 |
| --- | --- |
| 移动 | `W` `A` `S` `D` |
| 跳跃 | `空格` |
| 疾跑 | `Shift` |
| 观察 | 鼠标移动 |
| 挖掘方块 | 鼠标左键 |
| 放置方块 | 鼠标右键 |
| 选择草方块、泥土、石头、橡木 | 数字键 `1`–`4` |
| 切换快捷栏 | 鼠标滚轮 |
| 暂停/释放鼠标 | `Esc` |

## 特性

- Three.js r169，CDN 加载，无 npm 与构建步骤
- InstancedMesh 渲染的可编辑体素世界
- 程序生成起伏地形与橡树林
- 第一人称视角、重力、跳跃、碰撞和世界边界
- 准星选取、方块高亮、破坏粒子和简单音效
- Minecraft 风格像素纹理、快捷栏与生存 HUD
