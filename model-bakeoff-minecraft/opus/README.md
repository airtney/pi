# 迷你我的世界（Mini Minecraft）

用 Three.js 实现的第一人称体素方块沙盒，纯前端、无需构建、无需后端。直接用浏览器打开 `index.html` 即可游玩。

## 玩法

- 点击「开始游戏」，浏览器锁定鼠标进入第一人称视角。
- `W A S D` 移动，`空格` 跳跃，移动鼠标转动视角。
- `左键` 破坏准星对准的方块，`右键` 在准星方块相邻面放置方块。
- 数字键 `1`–`4` 或鼠标滚轮切换热键栏方块：草方块 / 泥土 / 石头 / 木头。
- `Esc` 释放鼠标指针，回到开始界面。

## 特性

- 程序化地形：value noise 生成小山与平地，含水面、沙滩与树木。
- 四种以上方块：草、泥土、石头、木头、树叶、水、沙（程序化像素贴图）。
- 物理系统：重力、跳跃、逐轴 AABB 碰撞，撞墙不穿模，掉出世界会重生。
- 交互：Amanatides & Woo 体素射线拾取，准星对准方块高亮描边。
- UI：十字准星、热键栏、破坏方块的粒子效果。
- 天空色 + 半球光/方向光/环境光（白天）+ 远处雾效。

## 文件

- `index.html` — 页面结构、importmap（CDN 引入 Three.js r160）、UI 元素。
- `style.css` — 十字准星、热键栏、开始遮罩等样式。
- `app.js` — 世界生成、网格构建、物理与碰撞、破坏/放置、控制与主循环。
- `README.md` — 本说明。

## 技术说明

- 依赖：Three.js `r160` 与 `PointerLockControls`，均通过 unpkg CDN 的 ES Module 加载。
- 世界为 56×56 的体素地图，只渲染临空可见面并按材质分组合批，编辑方块后重建网格。
- 需要联网加载 CDN；建议用本地静态服务器打开（如 `python3 -m http.server`）以避免部分浏览器对本地文件模块的限制。
