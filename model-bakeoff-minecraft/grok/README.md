# 迷你我的世界

用 Three.js 做的第一人称体素沙盒，打开即可游玩。

## 怎么玩

1. 用浏览器直接打开 `index.html`（需能访问 CDN）
2. 点击画面锁定鼠标，进入游戏
3. **WASD** 移动，**空格** 跳跃，鼠标转动视角
4. **左键** 破坏准星对准的方块，**右键** 放置
5. 数字键 **1–4** 切换热键栏：草方块 / 泥土 / 石头 / 木头
6. **Esc** 释放鼠标

## 文件

- `index.html` — 页面与 UI（准星、热键栏、开始菜单）
- `app.js` — 地形生成、树木、物理碰撞、破坏/放置
- `style.css` — 界面样式

## 技术

- Three.js r160（CDN，`importmap` + ES module）
- `PointerLockControls` 第一人称控制
- 程序噪声地形 + 树木 + 低洼水面
- 重力与 AABB 碰撞，不可穿墙

无需安装依赖或构建步骤。
